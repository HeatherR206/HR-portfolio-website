About Me

A child of the 1980s, the Apple IIe was my gateway into the tech world. For years, I loved using computers to learn new things and express myself creatively. Now, I'm leveling up, leaving teaching behind for a career in software engineering!

I'm passionate about technology that simplifies and enhances human experiences. I believe knowledge is power and am eagerly learning all I can about web development. My ever-expanding skill set includes UX design, typography, iconography, web design, user flow, and visual design.
