## **Heather's Developer Portfolio**

A responsive, professional portfolio website designed to showcase front-end development projects and UI/UX design capabilities.

**[View Live Site](https:heatherricarte-portfolio.netlify.app)**


## 🚀 Features
- **Responsive Design:** Seamless experience across all device sizes.
- **Interactive Contact Form:** Powered by Netlify Forms with custom AJAX handling.
- **Project Gallery:** Detailed breakdown of technical projects with skill tags.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Deployment:** Netlify (Continuous Integration)
- **Tools:** Git/GitHub, VS Code, Figma (Wireframing)
